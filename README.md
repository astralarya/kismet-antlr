# kismet-antlr

ANTLR Grammar for the Kismet family of dice roll parsers

## Setup

* [Install Gradle](https://gradle.org/install/)

## Usage

* Build default target (Java)

```
./gradlew build
```

* Run Antlr TestRig

```
./gradlew testrig
```

**NOTE:** Gradle may appear to hang at 80%,
but the TestRig is running
and accepting input
